# agentic-ai-projects

Hello,

We are currently living in a time where artificial intelligence is rapidly changing the world. AI is becoming more and more part of our daily lives.

Because of this change, I have become very interested in **Agentic AI**.

In this repository, I plan to build and share useful and interesting projects powered by Agentic AI.

My main goal is not only to write code directly, but to focus on how to use AI effectively to achieve specific goals. In other words, I want to learn how to guide and work with AI as a builder.

I will mainly use **Codex**, but depending on the project, I may also use other AI tools such as **Claude Code**, **GitHub Copilot**, or **Cursor**.

This repository is a place to explore how far we can go by working together with AI.
